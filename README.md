## Игра "Арифметическая прогрессия"
```
Показываем игроку ряд чисел, образующий арифметическую прогрессию, заменив любое из чисел двумя точками. Игрок должен определить это число.

 - Рекомендуемая длина прогрессии – 10 чисел. Длина может генерироваться случайным образом, но должна содержать не менее 5 чисел

 - Позиция спрятанного элемента каждый раз изменяется (выбирается случайным образом)
```

## Игра "Простое ли число?"
```
Простое число — это натуральное число больше единицы, которое делится без остатка только на 1 и на само себя (имеет ровно два различных натуральных делителя). Например, числа 2, 3, 5, 7, 11, 13 — простые, в то время как 4 (делится на 1, 2, 4) или 6 (делится на 1, 2, 3, 6) — нет. Число 1 не является простым, так как имеет только один делитель. 
```
